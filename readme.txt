Sport results, log for sportsmens to keep results, events description, positions, etc., etc.

== Description ==

Plugins allow to collect individual sports results.

== Installation ==

1. Upload Sport Results to your plugins directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure plugin using Results -> Settings
